# House Price Prediction

Predicting residential house prices using Regression models.

## Project Structure
- `data/`: Synthetic Housing dataset.
- `models/`: ML models.
- `plots/`: Analysis charts.
- `src/`: Code modules.

## Features
- **Area**: Square footage.
- **Location**: Neighborhood impact (Downtown, Suburb, etc.).
- **House Age**: Impact of depreciation.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Run the pipeline:
```bash
python main.py
```
